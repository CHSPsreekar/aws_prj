# Managing Cloud Object Storage with Amazon S3 via AWS Management Console

## Activity Overview
This activity demonstrates managing cloud object storage using **Amazon S3** via the **AWS Management Console**.

### Objective
Create two S3 buckets and put objects in a bucket through the AWS web console.

### Steps Performed
1. Logged into the **AWS Free Tier account**.
2. Opened the **S3** service from the AWS Management Console.
3. Created **two S3 buckets**:
   - `bucket-1`: Used for storing image files.
   - `bucket-2`: Used for storing document files.
4. Uploaded objects (e.g., images or text files) into `bucket-1`.
5. Verified bucket properties, permissions, and access settings.
6. Checked object details such as size, type, and storage class.
7. (Optional) Enabled versioning or logging for the bucket.

### Learnings
- Understanding of **Amazon S3** concepts such as buckets, objects, and storage classes.
- Practical knowledge of creating and configuring S3 buckets.
- Experience managing object permissions and visibility.

### Tools Used
- **AWS Management Console**
- **AWS Free Tier account**
